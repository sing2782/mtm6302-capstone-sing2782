Cosmic Explorer — Design & UX Report

Design & Branding Choices
Space Theme: The site uses a clean, dark "cinema" style. This removes bright screen glare so NASA’s colorful photos stand out as the main focus.
Modern Look: Instead of messy shadows, thin crisp borders are used to keep the layout looking flat, neat, and high-tech.

Colors & Font
Colors: Background is a soft midnight blue (`#0B0F19`) to prevent eye strain.
Content boxes use a slate blue (`#161F30`) to neatly separate sections.
Buttons use bright blue (`#3B82F6`) for searching and vibrant pink (`#EC4899`) for saving favorites.
Fonts: Titles are big and extra bold for a modern look. Description text has extra spacing between lines so long paragraphs are easy to read.

Layout & Responsiveness
The page dynamically rearranges itself depending on your device:
Mobile: Everything stacks in a single vertical column. Buttons stretch full-width so they are easy to tap with your thumb.
Tablet:The page widens and the saved favorites rearrange into a neat 2-column grid
Desktop:The main section splits into a side-by-side view (image on the left, text on the right) so you don't have to scroll. The favorites section expands into a column grid.

UX Improvements
One-Click Full Screen: Clicking the main image instantly dims the background and opens the high-def photo full-screen. Users can close it by clicking anywhere on the dark background, keeping them on the same page.
Organized Sections: Keeping the search form at the top and the saved collection at the bottom makes the app simple, clean, and easy to navigate.
